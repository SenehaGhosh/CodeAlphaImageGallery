# CodeAlpha Image Gallery

This is a responsive image gallery built using HTML, CSS, and JavaScript as part of my internship task.

# Features
- Responsive image grid layout
- Lightbox view for images
- Next and Previous navigation
- Hover effects with smooth transitions
- Image filters (All / Nature / City)

# Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open index.html in your browser
2. Click on images to view in lightbox
3. Use Next/Prev buttons to navigate
4. Use filter buttons to sort images

This project was created as part of my learning and internship experience.
